# Simple React Stopwatch

This is a simple and clean stopwatch project built with React and custom CSS. 
I designed it with a modern dark theme where everything sits in a perfect vertical line so it looks good on both phones and computers.

# How it Works
* **Start**: Starts counting up smoothly.
* **Stop**: Pauses the time exactly where it is.
* **Reset**: Sets the time back to 0.

# Tech Used
* React.js
* Clean CSS (Flexbox layout)

# How to Run It on Your Computer

1. Clone this project:
   ```bash
   git clone https://github.com
   ```

2. Go into the project folder:
   ```bash
   cd react-stopwatch
   ```

3. Install the packages:
   ```bash
   npm install
   ```

4. Start the app:
   ```bash
   npm start
   ```

Now open `http://localhost:3000` in your browser to see it run!
